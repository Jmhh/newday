
使用webapp Vue2.0+webpack+es6+vuex搭建的时光网电影webapp
====
简介:
---
该项目基于 douban_Website项目,尝试使用了Vue.js
  
Build Setup
---
```
# install dependencies
npm install

npm run dev

npm run build
```
